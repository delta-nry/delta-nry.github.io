[Tennis Mania](https://www.nateyee.com/TennisMania) is a game that I have been
working on in my spare time. I implemented it using the Unity game engine.

**Please note that you will need to use a desktop browser to play Tennis Mania,
as the Unity WebGL Player does not support mobile browsers at this time.**

Tennis Mania is based on the eponymous game in the book
*Hands-On Intro to Game Programming* by Chris DeLeon.
